# Asteroids Game

A classic Asteroids game built with Python and Pygame.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate  # On Windows: .venv\Scripts\activate
pip install pygame
```

## Run

```bash
python main.py
```

## Controls

- **A/D** - Rotate ship
- **W/S** - Move forward/backward
- **SPACE** - Shoot
